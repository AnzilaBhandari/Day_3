To check the configured username and user email in your Git configuration,

"git config --global user.name 'username'"
"git config --global user.email 'user email'"

To initialize new github repository,

"git init"

To check the status of repository,

"git status"

Staging the changes,

"git add ."

Commiting Changes,

"git commit -m "message" "

To see the commit history and log,

"git log"

To go to a specific commit,

"git commit <commit -hash>"
